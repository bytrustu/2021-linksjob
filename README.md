### 소개

원티드, 사람인, 크레딧잡, 잡플래닛, 로켓펀치 등 다양한 `채용 사이트의 링크` 를 한 곳에서 확인할 수 있도록 도와줍니다.  
관심 있는 기업의 채용 정보를 손쉽게 검색하고, 각 채용 사이트로 바로 이동할 수 있습니다. ✨

- 필요한 기업 정보의 링크 데이터 크롤링
- 기업 정보 시각화
- 개인 정보 관리
- 프로젝트 배포
- 반응형 작업


<br />

### 프로젝트 상세

- 기업의 링크 정보가 데이터베이스에 없을 경우, 해당 사이트를 크롤링하여 데이터를 수집합니다.
